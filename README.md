# Smart Campus Service Desk System

A full-stack ticket management platform built using Spring Boot and MySQL to streamline campus complaint workflows and service requests.

## Features
- Role-based authentication
- Ticket creation and tracking
- Department assignment
- Admin dashboard
- RESTful CRUD APIs
- Responsive Bootstrap UI

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL / H2
- Thymeleaf
- Bootstrap
- Maven
